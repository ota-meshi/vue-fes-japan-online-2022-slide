## `$children`、`$destroy()`もチェックする

プラグインではなく ESLint 本体に付属しているルールに、  
任意のプロパティをチェックできるルールがあるので、それを使用してチェックできます。

https://eslint.org/docs/latest/rules/no-restricted-properties

::left::

```js
module.exports = {
  // ...
  rules: {
    // ...
    "no-restricted-properties": [
      "error",
      {
        property: "$children",
        message: "廃止されたプロパティです",
      },
      {
        property: "$destroy",
        message: "廃止されたメソッドです",
      },
    ],
    // ...
  },
  // ...
};
```

::right::
