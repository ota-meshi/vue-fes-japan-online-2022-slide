## `import Vue from 'vue'`もチェックする

ESLint の標準ルールに、任意の`import`をチェックできるルールがあるので、  
それを使用してチェックできます。

https://eslint.org/docs/latest/rules/no-restricted-imports

::left::

```js
module.exports = {
  // ...
  rules: {
    // ...
    "no-restricted-imports": [
      "error",
      {
        paths: [
          {
            name: "vue",
            importNames: ["default"],
            message: "名前付きImportを使用してください。",
          },
        ],
      },
    ],
    // ...
  },
  // ...
};
```

::right::
